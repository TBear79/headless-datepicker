Calendar modes

Single array

Return dates as a single array by providing a start date and an end date.

Fill mode

Default behaviour. Always returns seven days for each week. If a date is not in the month, it fills it a null-values.

Adjacent mode

Always returns seven days for each week. If a date is not in the month, it still contains the date, but isAdjacent is set to true.

Exact mode

Doesn't return seven days in a week within the month. If a day is not in the current month, it is simply not part of the array.

Fixed mode

Always returns seven days for each week and it always returns six weeks. It fills the adjacent month dates with days.
